# Learning Rust by doing
This project allow me to learn more about async Rust and concurrency. 

## What is it doing ?
This is a little web scraper that you can pass to it two urls, it will get them, parse them and we will race them against each other to see the fastest site to respond.

### How to run this project ?
`$ cargo run -- https://www.rust-lang.org https://music.youtube.com/`
